Here is some stuff that will really help you out in the long run.

HackerRank is great for learning ins and outs of a language. Along with cpp, i'd really reccomend learning python as well.

LeetCode is great for interview prep and is really good for learning data structres. If you are going to use the "explore" section for data strctures (which I reccomend you should), make sure you do these first:

  1. Arrays and strings
  2. Queues and Stacks
  3. Recursion and Binary Search
  4. Linked lists
  5. Binary Trees

If you are interested in web development, check out FreeCodeCamp

If you are interested in competitive programming, such as ICPC , you should express your interest to either Professor Waxman or Edward Cerverizzo
